# GitHub Projects Automation

## Features
- Auto-add GitHub issues to Projects v2
- Auto-classify issues by label
- Status synchronization
- Simple logging

## Tech Stack
- Python
- GitHub Actions
- GitHub GraphQL API

## How to Use
1. Set PAT_TOKEN and PROJECT_ID in GitHub Secrets
2. Open a new issue → it will be automatically added to Project
3. Optional: add labels to set Type field
